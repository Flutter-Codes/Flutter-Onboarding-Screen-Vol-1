## Getting Started

#### 1. Copy or Download code in your system 

```
git clone https://github.com/Flutter-Codes/Flutter-Onboarding-Screen-Vol-1.git
```

#### 2. Got To Directry
```
cd Flutter-Onboarding-Screen-Vol-1
```
#### 3. Run Flutter App

```
flutter run
```
